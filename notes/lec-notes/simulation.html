Simulation¶

Suppose that $y_i \sim N[\theta_i, 1]$, where the scalar parameter $\theta_i$ varies across individuals with $\theta_i = \theta + u_i$, where $u_i$ representing unobserved heterogeneity that is assumed to have a known distribution. The density of y conditional on $u$ is simply

$$f(y_i|u_i;\theta)=\frac{1}{\sqrt{2\pi}}e^{\frac{-(y_i-\theta-u_i)^2}{2}}$$

To estimate $\theta$, we need to integrate out $u_i$. We assume that $u_i$ has a density

$$g(u)=e^{-u}e^{-e^{-u}}$$

The marginal density of $y$ is

$$f(y_i|\theta)=\int f(y_i|v;\theta) g(v) dv$$

which does not have a closed-form. Suppose we estimate $\theta$ using the MSL estimator. Your task is to show that the MSL works well by Monte Carlo simulations. You should conduct the simulations following the steps listed below (assume the true value of $\theta$ is 1).

  1. Generate a sample of $u_i$ and $y_i$, $i = 1, 2,\dots, N$.
  2. Simulate the marginal density $f(y_i|\theta)$.
  3. Estimate $\theta$ using the MSL estimator.

Requirements: set $N = 100, 500, 1000$. For each value of $N$, choose $S = 1, 10, 100, 1000, 10000$. In each of the $3 ∗ 5 = 15$ cases, report the estimate and its standard error.

$$\hat{\theta} = \text{argmax}_\theta \log \prod_{i=1}^N \frac{1}{S} \sum_{j=1}^S f(y_i;u_{ij},\theta) = \text{argmax}_\theta \sum_{i=1}^N \log \left[\frac{1}{S} \sum_{j=1}^S f(y_i;u_{ij},\theta)\right]$$

#  1. Generate a sample of u_i and y_i, i=1,2,...,N
def gen_u_INV(n, s=None):
    return -log( -log( uniform(size=(s, n) if s else n) ) )

@nb.vectorize(nopython=True)
def g(u):
    return exp(-u)*exp(-exp(-u))

@nb.jit(nopython=True)
def gen_u_MH(n, s, burn_in=1000, d=1):
    u = np.zeros((s, n))
    x0 = np.zeros(n)
    for c in range(s+burn_in):
        eta = uniform(-d, d, n)
        x1_tilde = x0 + eta
        prob = g(x1_tilde)/g(x0)
        prob[prob>1] = 1
        unif = uniform(0, 1, size=n)
        accept = unif < prob
        x1 = accept*x1_tilde + (1-accept)*x0
        x0 = x1
        if c >= burn_in:
            u[c-burn_in, :] = x1
    return u

def gen_u(n, s, method):
    assert method in ('INV', 'MH')
    if method == 'INV':
        return gen_u_INV(n, s)
    if method == 'MH':
        u = gen_u_MH(n, s if s else 1)
        return u if s else u.flatten()

def gen_y(u):
    return normal(1+u, size=u.shape)

# 2. Simulate the marginal density of f(y_i|theta)
def gen_lik(n, s, method):
    u = gen_u(n, s, method)
    y = gen_y(u)
    def lik(theta):
        f_hat = ( 1/sqrt(2*pi) * exp( -(y-theta-u)**2/2 ) ).mean(0)
        ll_neg = -log( f_hat ).sum()
        return ll_neg
    return lik

# 3. Estimate theta
def optimize(lik, init_guess=0, method='L-BFGS-B'):
    lik_model = minimize(lik, init_guess, method=method)
    theta_hat = lik_model['x'].item()
    return theta_hat

# Main simulation function
def main(n, s, samp_method='INV', op_method='L-BFGS-B', init_guess=0):
    lik = gen_lik(n, s, samp_method)
    theta_hat = optimize(lik, init_guess, op_method)
    return theta_hat
